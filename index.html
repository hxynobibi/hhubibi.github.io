git报错 fatal:open /dev/null or dup failed: No such file or directory

大概是一两周前装的git for windows，然后搞了个博客。装的时候已经把C:\Windows\System32\drivers里面的null.sys替换掉了。

今天想要用git的时候发现git bash here打不开，cmd上git --version也出错了，报的错是fatal:open /dev/null or dup failed: No such file or directory

尝试了很多种方法

  • 卸载git，重新装，没用，而且git卸载不干净
  • 修改注册表里面\HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\Null第一个数据值为1，然后重启，没用
  • 管理员身份打开cmd，输入命令sfc /scannow，检查修复系统出现问题到100%，重启，终于成功了

是真的坑
