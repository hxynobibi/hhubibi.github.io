另外常用的一些命令：

  • git status 可以得到git现在的状态，是否是add了但是没有commit
  • git log 提交记录，类似于日志
  • git show ID 查看提交的一些内容，提交者等信息
  • git checkout ID [file] 将文件内容反转为之前提交的东西，需要再次commit

注意事项：

  • 第一个git pull可能会出现“fatal: refusing to merge unrelated histories”错误，选下一个git pull命令
